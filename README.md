# STM32-Based Pillbox Temperature and Humidity Alarm System

本项目基于 STM32F103C8 单片机，设计并实现了一套药盒温度与湿度实时检测与报警系统。系统通过 DHT11 传感器采集环境温湿度数据，采用 OLED 显示屏实时显示数值，并结合蜂鸣器和状态指示灯实现异常预警。

## 项目简介

该系统致力于为药品存储环境提供稳定的温湿度监测与预警功能，适用于小型药盒、医药柜或药房展示环境。项目重点包括数据采集、OLED 实时显示、串口通信以及异常报警逻辑。

## 核心功能

- DHT11 温湿度采集
- OLED 实时数据显示
- 异常温度报警提醒
- 蜂鸣器报警输出
- LED 状态指示
- 串口输出温湿度数据

## 硬件平台

- STM32F103C8 开发板
- DHT11 温湿度传感器
- OLED 显示屏
- 蜂鸣器
- LED 指示灯
- 串口调试工具

## 软件环境

- Keil uVision5
- STM32 标准外设库
- C 语言开发

## 项目结构说明

- `Main.c`：系统主程序，负责传感器初始化、数据读取、显示更新和报警控制
- `oled.c / oled.h`：OLED 显示模块驱动
- `Usart.c / Usart.h`：串口初始化与数据输出
- `speak.c / speak.h`：蜂鸣器控制模块
- `LED.c / LED.h`：LED 指示灯控制模块

## 使用说明

1. 使用 Keil uVision5 打开工程文件。
2. 确认开发板与 DHT11、OLED、蜂鸣器和 LED 正确连接。
3. 编译并下载程序到 STM32 开发板。
4. 通过串口调试工具观察温湿度数据输出。

## 适用场景

- 药盒温湿度监控
- 医药存储环境检测
- 小型电子报警系统演示
